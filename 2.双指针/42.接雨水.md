给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

 

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/10/22/rainwatertrap.png)

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
```

**示例 2：**

```
输入：height = [4,2,0,3,2,5]
输出：9
```

 

**提示：**

- `n == height.length`
- `1 <= n <= 2 * 104`
- `0 <= height[i] <= 105`



```
想象两个指针 left 和 right 分别从两头向中间靠拢。同时维护两个变量 left_max 和 right_max，记录当前左右两边分别出现过的最高柱子高度。

1.木桶效应：位置 $i$ 能接的水量等于 min(left_max, right_max) - height[i]。

2.移动逻辑：
如果 left_max < right_max，说明左边是“短板”。此时左边位置能接多少水只取决于 left_max（因为右边一定有更高的柱子挡着）。
反之，则处理右边。
```



```python
class Solution:
    def trap(self, height: List[int]) -> int:
        if not height: return 0
        
        left, right = 0, len(height) - 1
        left_max, right_max = 0, 0
        res = 0
        
        while left < right:
            # 更新左右两边的最高高度
            left_max = max(left_max, height[left])
            right_max = max(right_max, height[right])
            
            # 核心：哪边低，就处理哪边
            if left_max < right_max:
                # 左边是瓶颈，计算当前 left 指针处的盛水量
                res += left_max - height[left]
                left += 1
            else:
                # 右边是瓶颈，计算当前 right 指针处的盛水量
                res += right_max - height[right]
                right -= 1
                
        return res
```
