#### 题目描述



给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

示例：

```
输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
```



#### 核心思路



```
如果一个数字 n 是序列的起点，那么 n-1 一定不在集合中。
1.如果 n-1 在集合里，说明 n 只是某个序列中间的一个数，我们直接跳过它（等遍历到真正的起点时再处理）。

2.如果 n-1 不在集合里，说明 n 是起点，我们开始往后数 n+1, n+2, ... 直到断开为止。
```



#### 代码



```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        # 1. 将数组转为集合，实现 O(1) 复杂度的查找，同时去重
        nums = set(nums)
        max_length = 0

        for num in nums:
            # 2. 核心判断：只有当 num 是序列的“起点”时，才开始计数
            # 如果 num - 1 在集合中，说明 num 不是起点，跳过
            if (num - 1) not in nums:
```
